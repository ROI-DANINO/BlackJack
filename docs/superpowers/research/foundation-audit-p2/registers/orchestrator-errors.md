Gate criterion 10 requires the gate summary to disclose orchestrator errors. Phase 1 disclosed two,
and that disclosure is the only reason they are known. This file is the running record so the
disclosure is assembled from contemporaneous notes rather than from recollection at gate time.

An error is logged here whether or not it changed an outcome. "It did no harm" is a finding about
the error, not a reason to omit it.
